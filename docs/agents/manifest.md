# Manifesto de Agentes

## Formato de agente

Cada agente deve possuir:

- id
- nome
- cargo
- missão
- departamento
- ferramentas permitidas
- ações proibidas
- nível de autonomia
- critérios de qualidade
- gates de validação

## Agentes oficiais do MVP

- aninha
- byte
- lala
- dinkwink
- fiscal
- raspador
- traffic
- call
- suporte
- controller
- juridico
- documentador
