# Matriz de Risco

## Baixo

- Ler arquivo.
- Resumir documento.
- Criar plano.
- Criar documentação.

## Médio

- Alterar arquivo local.
- Criar script.
- Rodar comando.
- Modificar código.

## Alto

- Commit.
- Push.
- Deploy.
- Envio externo.
- Campanha paga.
- Banco de dados.

## Crítico

- Apagar arquivos.
- Acessar secrets.
- Alterar produção.
- Transação financeira.
- Compromisso jurídico.

## Política

Alto e crítico exigem aprovação humana explícita.
