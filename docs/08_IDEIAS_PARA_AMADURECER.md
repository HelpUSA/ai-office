# 08 - Ideias para Amadurecer

## Ideias principais

1. Mapa visual do escritório.
2. Cartões dos agentes.
3. Botão delegar.
4. Reunião simulada entre agentes.
5. Memória de decisões.
6. Score de autonomia.
7. Simulador de custo.
8. Loja interna de agentes.
9. Auditor de promessas comerciais.
10. Treinamento contínuo com erros reais.

## Score de autonomia

- Nível 0: só responde.
- Nível 1: lê e resume.
- Nível 2: sugere.
- Nível 3: cria arquivos seguros.
- Nível 4: executa comandos seguros.
- Nível 5: executa ações externas com aprovação.
- Nível 6: autonomia parcial auditada.
